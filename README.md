# Quenade

AI-powered local coding assistant.

## Features

- **Code explanation:** Entiende qué hace cada línea de tu código.
- **Code review:** Recibe sugerencias para mejorar la calidad.
- **Security analysis:** Detecta posibles fallos de seguridad.
- **Local LLM support:** Privacidad total (tus datos no salen de tu PC).

## Stack

- **Language:** Python
- **Engine:** Ollama
- **Model:** Qwen
- **OS:** Linux

## Examples

Para usar Quenade, ejecuta los siguientes comandos en tu terminal:

```bash
python3 main.py explain file.py
python3 main.py review file.py
python3 main.py security file.py
